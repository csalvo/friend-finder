# Friend Finder 
UMass Amherst Coding Boot Camp Assignment Week #13

### [Live App](https://bearybestfriendfinder.herokuapp.com)

### Overview

A compatibility-based "FriendFinder" application that takes in results from users' surveys, then compares answers with those from other users and displays the name and picture of the user with the best overall match. 

### What it's built with

1. Node 
2. Express
3. HTML/CSS
